Redis ZRANGEBYLEX demo

WARNING: The demo is turned off currently for lack of free memory in the small VPS used by the redis.io web site. We plan to activate it at least in a reduced form in the next days.

This is a demo of the new ZRANGEBYLEX Redis command. It autocompletes the query field using all the 1514 unique function names of the Redis source code as autocompletion dictionary.

Every time the user types something, the page sends a request to a small Ruby snippet, which sends a query to Redis and returns the result to the web page as a JSON array.
As you type in the input field below, you'll see the list of suggestions appearing.

The javascript was adapted from an example taken from JQuery Autocomplete.
